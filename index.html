Settings

Google Drive
Connect your Google Drive account to save all data — expense records, receipts, photos, and audio — permanently in the cloud. Works across all your devices.
One-time setup (5 minutes):
  1. Go to console.cloud.google.com, sign in with the new Google account you created.
  2. Click Select a Project → New Project. Name it GharBanao.
  3. Go to APIs & Services → Library. Search Google Drive API and enable it.
  4. Go to OAuth consent screen. Choose External. Fill app name GharBanao and your email.
  5. Go to Credentials → Create Credentials → OAuth Client ID. Choose Web application. Add your site URL to Authorised JavaScript origins. Copy the Client ID.
  6. Paste it below and click Connect.

All receipts, audio, and data sync to your GharBanao folder.

How to deploy (free)

GitHub Pages FreeEasiest

Upload this index.html to a GitHub repository. Go to Settings → Pages → set source to main branch. Your URL will be https://yourname.github.io/gharbanao. Done. No coding needed — just drag and drop the file.

Netlify Drop Free

Go to app.netlify.com/drop and drag the index.html file onto the page. It deploys instantly with a public URL. No account needed.
